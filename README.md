## ðŸ“– About The Project

**Omega LIS** is a lightweight, standalone medical laboratory system designed to demonstrate the power of mobile edge computing. Created during a "New Year's Coding Challenge" (Jan 1, 2026), this project proves that enterprise-grade logicâ€”including IoT simulation, PDF reporting, and triage algorithmsâ€”can be deployed without cloud infrastructure, using just a Python environment on a mobile device.

**Key Architecture:**
* **Server:** An Android smartphone running Python (Flask) acts as the Intranet Server.
* **Client:** Tablets/Laptops connect via local Hotspot to access the Doctor's Dashboard.
* **IoT Layer:** Background threads simulate biochemistry analyzers sending HL7-like data.

## ðŸš€ Features

* **ðŸ” Secure Access:** Session-based authentication system.

## ðŸ“¦ Installation & Usage

### Prerequisites
* Python 3.x
* Android device with Pydroid 3 OR any PC with Python.

### Running Locally

1.  Clone the repo:
    ```bash
    git clone [https://github.com/YOUR_USERNAME/omega-lis-pocket.git](https://github.com/YOUR_USERNAME/omega-lis-pocket.git)
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the application:
    ```bash
    python main.py
    ```
4.  Open your browser at `http://localhost:5000` (or your local IP).
    * **Login:** `admin`
    * **Password:** `omega`
